在Ubuntu下使用PHP连接SQL Server

Apr 11, 2014 #Ubuntu #PHP #SQL Server

最近我需要实现在Ubuntu系统下使用PHP连接SQL Server数据库。查看了PHP的官方文档并google了关键字之后，找到了一些相关资料，最终实现了目标！

系统配置：

  • Ubuntu 13.10
  • Microsoft SQL Server 2008 R2

实现过程： 第一步、安装Apache和PHP 在终端下分别使用命令sudo apt-get install apache2和sudo apt-get install php5安装Apache和PHP，使用phpinfo()检查是否安装成功。

第二步、安装FreeTDS及相关组件 使用以下命令安装FreeTDS及相关组件sudo apt-get install freetds-common freetds-bin unixodbc php5-sybase，安装成功之后，使用命令sudo service apache2 restart重启Apache。

现在你可以使用tsql命令测试连接SQL Server数据库了，在终端下输入：

    tsql -H <hostname/ip_address> -p <port> -U <username> -P <password> -D <chosen_database>

你应该会得下类似下面的信息：

    locale is "en_US.UTF-8"
    locale charset is "UTF-8"
    using default charset "UTF-8"
    Default database being set to <chosen_database>
    1> quit

如果你看到了类似上面的东西，那就说明连接成功了。接下去该修改/etc/freetds/freetds.conf文件了，这是FreeTDS的配置文件，使用编辑器打开这个文件，你会看到一些默认的内容，在文件末尾加上类似下面的内容：

    [YOUR_SERVER_FRIENDLY_NAME]
        host = <hostname/ip_address>
        port = <port>
        tds version = 8.0

现在你应该无需提供主机名和端口号就可以连接了，使用下面的命令测试：

    tsql -S <YOUR_SERVER_FRIENDLY_NAME> -U <username> -P <password> -D <chosen_database>

应该会得到类似下面的信息：

    locale is "en_US.UTF-8"
    locale charset is "UTF-8"
    using default charset "UTF-8"
    Default database being set to <chosen_database>
    1> quit

如果你得到了上面的信息，说明成功了，开始准备编写PHP代码吧！可以参考以下代码进行测试：

    $connection = mssql_connect('<YOUR_SERVER_FRIENDLY_NAME>', '<username>', '<password>');
    if (!$connection) {
      die('Unable to connect!');
    }
    if (!mssql_select_db('<chosen_database>', $connection)) {
      die('Unable to select database!');
    }
    $result = mssql_query('SELECT * FROM <some_table>');
    while ($row = mssql_fetch_array($result)) {
      var_dump($row);
    }
    mssql_free_result($result);

在浏览器中运行这些代码后，应该会打印出some_table表中的全部数据。
